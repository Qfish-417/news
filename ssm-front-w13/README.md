# Vue 3 + Vite

## 淘宝最新镜像源
npm config set registry https://registry.npmmirror.com

## 安装 element-plus
npm install element-plus

## 安装 vue-router
npm install vue-router

## 修改根目录下的index.html文件
## 修改 src 文件夹下style.css、main.js、App.vue
## 在 src 文件夹下新建 router 文件夹，添加路由文件index.js
## 在 src 文件夹下新建 views 文件夹，添加和路由对应的页面文件

## 安装 axios
npm install axios

## 在 src 文件夹下创建 service 文件夹，并在里面创建 index.js 文件，封装 axios 请求方法
## 同时在 service 文件夹下新建 request.js 文件
## 在 src 文件夹下创建 api 文件夹，并创建 index.js 文件写请求接口
## 在异步请求页面中调用 api 中的接口方法，发起起步请求
## 配置代理， 修改根目录下的 vite.config.ts 配置文件

## 安装 pinia 和 持久化工具 pinia-persistedstate-plugin
npm install pinia
npm install pinia-persistedstate-plugin

## 安装 vite插件
npm install @vitejs/plugin-vue
